“回馈家乡”是 加拿大「亚基思国际基金会」(Acacia Spring Charitable Foundation）与中国广西慈善总会合作举办的一个爱心公益项目。其基本理念是「施比受更为有福」，鼓励曾接受基金会资助或是有志服务家乡的农村大学青年，不忘根本，学会感恩，每年暑假期间返回家乡，服务有需要的群体。 首届 “回馈家乡”于2009 年由文更国际基金会与广西慈善总会合办。2012年，文更国际基金会更名为「亚基思国际基金会」。2014年，亚基思国际基金会正式与中国儿童少年基金会展开合作，共同推进中国慈善事业的发展。

活动影响

更多 >

“回馈家乡” 从2009年至今、已覆盖桂林阳朔、资源、恭城、龙胜、灵川、灌阳、柳州融水、百色隆林等地，活动参与大学生人数每年超过一百多名、受益儿童一千多名，人数逐年递增，深得当地贫困山区村民和社会各界人士的表扬和支持，影响深远。
